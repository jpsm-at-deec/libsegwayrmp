Sets the Callback Function to be called on when a timestamp is made.

This allows you to provide your own time stamp method for time stamping the segway data. You must return a Segway\+Time\+Struct, but you can return an empty time struct and manually store your time stamp and use your timestamp when processing the segway status callback.
